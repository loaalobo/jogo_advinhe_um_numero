# jogo_advinhe_um_numero

Esse exercício foi um desafio proposto no curso de lógica básica de programação da @alura.

Segue o desafio:
1 - criar uma função sorteia que recebe um número min e max e sorteia um número entre esses valores, retornando esse valor. 
2 - fazer o jogo mostrar, quando o usuário errar a tentativa, se o número que ele chutou era maior ou menor ao número pensado pelo programa.

Tecnologias utilizadas: html, css
Framework: bootstrap
Linguagem de programação: javascript
